◆ uart_receive_dma()

void uart_receive_dma ( uart_num_e  uart_num,
unsigned char *  addr,
unsigned char  rev_size 
)

This function serves to receive data function by DMA, this function tell the DMA to get data from the uart data fifo.

Parameters
[in]uart_num- UART0 or UART1.
[in]addr- pointer to the buffer receive data.
[in]rev_size- the receive length of DMA
Note
The DMA version of A0 has some limitians. 1:The receive length should be greater or equal to the data you want to receive,then the data won't be lost. 2:You have to estimate the data-length that you want to receive.If the data length you set isn't the multiple of 4(the DMA carry 4-byte one time),like 5,it will carry 8 byte,while the last 3-byte data is random.
Returns
none
