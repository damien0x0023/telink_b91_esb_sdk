◆ clock_init()

_attribute_ram_code_sec_noinline_ void clock_init ( sys_pll_clk_e  pll,
sys_clock_src_e  src,
sys_pll_div_to_cclk_e  cclk_div,
sys_cclk_div_to_hclk_e  hclk_div,
sys_hclk_div_to_pclk_e  pclk_div,
sys_pll_div_to_mspi_clk_e  mspi_clk_div 
)

This function use to select the system clock source.

Parameters
[in]pll- pll clock.
[in]src- cclk source.
[in]cclk_div- the cclk divide from pll.it is useless if src is not PAD_PLL_DIV. cclk max is 96M
[in]hclk_div- the hclk divide from cclk.hclk max is 48M.
[in]pclk_div- the pclk divide from hclk.pclk max is 24M.
[in]mspi_clk_div- mspi_clk has two source. pll div and hclk.mspi max is 64M.
Returns
none
