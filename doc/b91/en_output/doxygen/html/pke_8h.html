◆ pke_check_rt_code()

static unsigned char pke_check_rt_code ( void  )
inlinestatic

This is used to indicate the reason when the pke stopped.

Returns
0 - normal stop. 1 - received a termination request(CTRL.STOP is high). 2 - no valid modulo inverse. 3 - point is not on the curve(CTRL.CMD:PVER). 4 - invalid microcode.
