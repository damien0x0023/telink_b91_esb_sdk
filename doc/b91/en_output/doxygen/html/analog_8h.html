void analog_read_buff_dma ( dma_chn_e  chn,
unsigned char  addr,
unsigned char *  pdat,
unsigned int  len 
)

This function write buffer to analog register by dma channel.

Parameters
[in]chn- the dma channel.
[in]addr- address need to be read from.
[out]pdat- the buffer ptr to store read data. note: The size of array pdat must be a multiple of 4. For example, if you just need read 5 byte by dma, you should define the size of array pdat to be greater than 8 other than 5. Because the dma would return 4 byte data every time, 5 byte is not enough to store them.
[in]len- the length of read data.
Returns
none.

◆ analog_write_addr_data_dma()

void analog_write_addr_data_dma ( dma_chn_e  chn,
void *  pdat,
int  len 
)

This function write buffer to analog register by dma channel.

Parameters
[in]chn- the dma channel.
[in]pdat- the buffer(addr & data) ptr need to be write, note: The array pdat should look like this,
pdat
pdat[0] address 0x3a
pdat[1] data 0x11
pdat[2] address 0x3b
pdat[3] data 0x22
......
It means write data 0x11 to address 0x3a, write data 0x22 to address 0x3b, ......
[in]len- the length of read data.
Returns
none.
